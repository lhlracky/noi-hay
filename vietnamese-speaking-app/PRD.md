# 越南语口语练习 App — 产品需求说明 (PRD)

## 1. 产品概述

### 1.1 产品名称
**Nói Hay**（越南语"说得好"）

### 1.2 产品愿景
让中文母语者通过沉浸式口语练习，自信开口说越南语。

### 1.3 目标用户
| 用户画像 | 描述 |
|---------|------|
| 旅行者 | 计划去越南旅游，需要基础日常会话 |
| 商务人士 | 在越南工作/经商，需要商务沟通能力 |
| 语言爱好者 | 对越南语感兴趣，想系统学习 |
| 留学生 | 在越南学习，需要提升口语流利度 |

### 1.4 核心价值主张
- **AI 驱动**：24/7 随时练，不需要找语伴
- **即时反馈**：发音评分 + 纠正建议，说完就知道对不对
- **场景化**：真实生活场景，学了就能用
- **渐进式**：从单词 → 短句 → 对话，难度递增

## 2. 功能需求

### 2.1 MVP（最小可行产品）

#### 2.1.1 语音录入与识别
- 用户按下录音按钮说出越南语
- 实时波形动画反馈（让用户知道在录）
- 支持最长 30 秒连续语音
- 自动检测静音并截断

#### 2.1.2 发音评估
- **总分**：0-100 分
- **维度评分**：
  - 音调准确度（越南语 6 声调是核心难点）
  - 元音/辅音准确度
  - 语速与节奏
  - 完整度（是否漏词）
- **可视化反馈**：
  - 每个词/音节的颜色标记（绿=好，黄=一般，红=需改进）
  - 声调曲线对比图（用户 vs 标准）

#### 2.1.3 AI 对话练习
- 预设场景卡片（15+ 场景）
  - 🍜 餐厅点餐
  - 🚕 打车/问路
  - 🛒 市场砍价
  - 🏨 酒店入住
  - 🏥 看医生
  - 📞 打电话
  - 👋 自我介绍
  - ☕ 咖啡馆闲聊
  - ……
- AI 扮演对话角色（服务员、司机、店主等）
- 每轮对话后给出表达建议和替代说法
- 支持"请说慢一点"和"请重复"指令

#### 2.1.4 学习路径
- 分级：A1（入门）→ A2（基础）→ B1（中级）→ B2（中高级）
- 每级包含：词汇 → 常用句 → 场景对话
- 解锁机制：完成当前级别 80% 后解锁下一级

#### 2.1.5 个人中心
- 练习天数 / 总时长
- 各场景完成度
- 发音分数趋势图
- 收藏的句子/词汇

### 2.2 V2 功能（后续迭代）

| 功能 | 说明 |
|------|------|
| 语音克隆 | 用户可以听自己的发音回放 vs 标准发音 |
| 每日挑战 | 每天一个小任务（说一句话/完成一个场景） |
| 社交排行 | 和其他学习者比拼练习时长/分数 |
| 离线模式 | 核心功能无需联网 |
| 越南语打字 | 学习越南语输入法，拼写练习 |
| 文化小贴士 | 场景对话中穿插越南文化知识 |

## 3. 非功能需求

### 3.1 性能
- 语音识别延迟 < 2 秒
- 发音评分延迟 < 3 秒
- 页面加载 < 1.5 秒
- 支持弱网环境（语音异步处理）

### 3.2 平台
- **一期**：Web App（响应式，手机优先）
- **二期**：iOS / Android 原生 App

### 3.3 语言
- 界面语言：中文
- 学习语言：越南语（北部标准音为主，可选南部音）

### 3.4 可用性
- 无需注册即可体验（游客模式）
- 支持微信/手机号登录
- 无障碍：支持大字体模式

## 4. 用户旅程

```
首次打开
  │
  ├─ 欢迎页 → 选择学习目标（旅行/工作/兴趣）
  │            → 测试当前水平（简短口语测试）
  │            → 推荐起始级别
  │
  ├─ 主界面
  │   ├─ 今日推荐（3个练习卡片）
  │   ├─ 场景列表（按主题分类）
  │   └─ 个人进度
  │
  └─ 练习流程
      │
      ├─ [词汇] 听标准发音 → 跟读 → 评分 → 纠正
      │
      ├─ [句子] 听句子 → 跟读 → 评分 → 对比
      │
      └─ [对话] 场景介绍 → AI说第一句 → 用户说
                → AI回复 → 用户说 → ... → 结束
                → 总结（分数 + 建议 + 新词汇）
```

## 5. 竞品分析

| 产品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|-------------|
| Duolingo | 游戏化、用户多 | 越南语口语弱，无发音评分 | 专注口语+发音评估 |
| Speechling | 发音纠正好 | 无越南语 | 越南语深度覆盖 |
| Pimsleur | 听说训练系统 | 贵、无AI互动 | AI对话+免费基础功能 |
| ChatGPT语音 | 灵活 | 无结构化课程、无评分 | 结构化学习路径+评分系统 |

## 6. 成功指标

| 指标 | MVP 目标 | 说明 |
|------|---------|------|
| 日活用户 (DAU) | 500 | 上线 30 天 |
| 平均练习时长 | 8 分钟/天 | 单次会话 |
| 次日留存率 | 40% | 新用户次日回访 |
| 发音分数提升 | +15 分 | 使用 2 周后平均提升 |
| 场景完成率 | 60% | 开始场景到完成 |

## 7. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 越南语 ASR 准确率不足 | 识别错误导致评分不准 | 多模型对比，收集反馈优化 |
| 声调评估难度大 | 核心功能质量差 | 专项训练声调模型 |
| 用户坚持不下来 | 留存差 | 游戏化机制 + 推送提醒 |
| 服务器成本高 | 预算超支 | 语音处理限频 + 缓存 |
